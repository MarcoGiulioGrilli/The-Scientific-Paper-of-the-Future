# Factorial Calculator

Seminar 3 – *The Scientific Paper of the Future*  
Best Practices for Sharing Data, Software and Methods when writing Research Reports.

## Description

This repository contains a simple C program that calculates the factorial of a non-negative integer using recursion.

The purpose of this repository is to demonstrate best practices for sharing research software.

## Requirements

- GCC compiler
- C standard library

## Compilation

Compile the program using GCC:

```bash
gcc factorial.c -o factorial
